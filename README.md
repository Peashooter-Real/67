# 🎮 เกมนับ 7 ออนไลน์ (Count to 7 Online)

เกมไพ่วัดสติและความไว เล่นออนไลน์กับเพื่อนได้ทันที!

## 🚀 วิธีการติดตั้งและรันเครื่อง (Local)
1. ติดตั้ง Node.js
2. เปิด Terminal ในโฟลเดอร์นี้
3. พิมพ์ `npm install`
4. พิมพ์ `npm start`
5. เข้าไปที่ `http://localhost:3001`

---

## 🌍 วิธีการเปิดให้เพื่อนเล่นผ่านลิ้งค์ (Deployment)
เนื่องจากเกมนี้มีระบบ Backend สำหรับเล่นหลายคน (Multiplayer) จะต้องใช้วิธีการฝากไฟล์ที่รองรับ Node.js (แนะนำ **Render.com** เพราะฟรีและง่ายที่สุด)

### 1. อัพโหลดขึ้น GitHub
1. สร้าง Repository ใหม่ใน GitHub ของคุณ
2. อัพโหลดไฟล์ทั้งหมดขึ้นไป (ยกเว้นโฟลเดอร์ `node_modules`)
   - `index.html`, `lobby.html`, `game.html`
   - `server.js`
   - `assets/` (รูปภาพ)
   - `package.json`
   - `.gitignore`

### 2. เชื่อมต่อกับ Render.com (ฟรี)
1. สมัครใช้งานที่ [Render.com](https://render.com/)
2. กด **New +** และเลือก **Web Service**
3. เชื่อมต่อกับ GitHub ของคุณและเลือก Repository "นับ 7"
4. ตั้งค่าดังนี้:
   - **Runtime**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
5. กด **Create Web Service**
6. เมื่อระบบรันเสร็จ คุณจะได้ลิ้งค์มา (เช่น `https://count7.onrender.com`) ส่งลิ้งค์นี้ให้เพื่อนเล่นได้เลย!

## 🃏 กฎการเล่น
- รวมกอง 100 ใบ แบ่งให้เท่ากัน
- นับ 1-7 แล้ววนกลับ 7-1 ไปเรื่อยๆ
- **คนธรรมดา**: นับเลขตามลำดับ
- **รูปโทรศัพท์**: พูดว่า "HELLO"
- **รูปปืน**: "เงียบ" (SILENCE)
- **รูปคู่ (x2)**: ทำแอคชั่นเดิม 2 ครั้ง และข้ามคนถัดไป

ขอให้สนุกกับการนับ! 🏆💨
